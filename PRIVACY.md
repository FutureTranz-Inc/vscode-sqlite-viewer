# Privacy Policy

**Effective Date:** January 18, 2026

## Overview

FutureTranz Inc. ("we", "us", "our") respects your privacy. This Privacy Policy describes how the **SQLite Viewer** Visual Studio Code extension handles your data.

## Data Collection

**We do not collect, store, or transmit any personal data or database content.**

- **Local Operation**: The extension operates entirely locally on your machine.
- **No Telemetry**: We do not include any custom telemetry or tracking scripts in this extension.
- **Database Access**: The extension reads SQLite files solely for the purpose of displaying them within the VS Code editor. No data is sent to external servers.

## VS Code Telemetry

This extension runs within Visual Studio Code. Microsoft may collect telemetry data according to their privacy statement. You can disable VS Code telemetry in your settings.

- [Visual Studio Code Privacy Statement](https://code.visualstudio.com/privacy)

## Changes to This Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page.
