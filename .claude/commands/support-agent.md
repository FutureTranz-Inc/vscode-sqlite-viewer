# Support Agent

Consult the Support agent for customer support ticket management, technical issue resolution, and knowledge base development.

**Primary Use Cases:**
- Customer support ticket management
- Technical support issue resolution
- Support process optimization
- Knowledge base development
- Customer satisfaction tracking
- Escalation management

**When to Use:**
- Support ticket analysis
- Response template creation
- Support process improvements
- Customer satisfaction initiatives
- Escalation procedures

**Examples:**
```
Use support-agent to analyze support tickets for common authentication issues with resolution recommendations
Use support-agent to develop knowledge base article for database connection troubleshooting
Use support-agent to optimize support process for enterprise client onboarding workflow
```

**Quick Start:**
1. Describe the support scenario or tickets
2. Provide customer context and issue details
3. Include current support processes
4. Specify improvement objectives
