# UI/UX Designer Agent

Consult the UI/UX Designer agent for component design, accessibility audits, responsive design, and user experience optimization.

**Primary Use Cases:**
- UI component design
- UX flow analysis
- Accessibility audits (WCAG)
- Responsive design
- Design system creation
- UI performance optimization

**Available Functions:**
- `design_component` - Create/improve UI components with HTML, CSS, and UX considerations
- `analyze_ux_flow` - Analyze user experience flows and suggest improvements
- `accessibility_audit` - Perform WCAG accessibility audits on components
- `responsive_design` - Create responsive design solutions for multiple screen sizes
- `design_system` - Create/enhance design system components and guidelines
- `optimize_performance` - Optimize UI performance and loading experience

**When to Use:**
- Building new UI components
- Accessibility requirements
- Multi-device support needed
- Performance optimization for frontend
- Design system development

**Examples:**
```
Use ui-ux-designer agent to design_component for accessible navigation menu with WCAG AA compliance and mobile-first approach
Use ui-ux-designer agent to analyze_ux_flow for e-commerce checkout process with conversion optimization
Use ui-ux-designer agent to responsive_design for dashboard interface supporting desktop, tablet, and mobile
```

**Quick Start:**
1. Describe the UI component or flow to design
2. Provide user context and use cases
3. Include accessibility requirements (WCAG level)
4. Specify target devices/platforms
