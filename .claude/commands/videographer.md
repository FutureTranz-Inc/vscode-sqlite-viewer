# Videographer Agent

Consult the Videographer agent for video content strategy, production planning, editing, and platform optimization.

**Primary Use Cases:**
- Video content strategy
- Production planning and storyboarding
- Video editing and post-production
- Motion graphics and animation
- Video optimization for platforms
- Tutorial and demo video creation

**When to Use:**
- Product demos
- Tutorial content
- Marketing videos
- Social media video content
- Event coverage

**Examples:**
```
Use videographer agent to plan product demo video for SaaS platform with customer testimonial integration
Use videographer agent to create storyboarding for explainer video on complex technical concept
Use videographer agent to optimize video content for YouTube SEO and platform algorithm
```

**Quick Start:**
1. Describe the video content objective
2. Provide target audience and platform
3. Include video length and style preferences
4. Specify production constraints or resources
