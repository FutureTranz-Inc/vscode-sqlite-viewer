# Time & Billing Agent

Consult the Time & Billing agent for timesheet management, project cost tracking, invoicing, and profitability analysis.

**Primary Use Cases:**
- Timesheet management and tracking
- Project cost tracking and billback
- Resource utilization analysis
- Invoice generation and billing
- Profitability analysis
- Budget variance tracking

**Available Functions:**
- `track_time` - Log time entry with project allocation and billing details
- `generate_timesheet` - Generate comprehensive timesheet report for specified period
- `calculate_project_costs` - Calculate total project costs with labor and expense breakdown
- `analyze_utilization` - Analyze resource utilization and capacity planning
- `generate_invoice` - Create client invoices with detailed billing breakdown
- `track_profitability` - Analyze project profitability with margin reporting

**When to Use:**
- Project billing and cost tracking
- Resource capacity planning
- Client invoicing
- Profitability analysis
- Time compliance audits

**Examples:**
```
Use time-billing agent to generate_invoice for Q4 2024 consulting services with detailed time breakdown
Use time-billing agent to analyze_utilization for development team with capacity planning recommendations
Use time-billing agent to track_profitability for mobile app project with margin variance analysis
```

**Quick Start:**
1. Specify the billing period or project
2. Provide time tracking data or requirements
3. Include billing rates and client terms
4. Mention any special billing arrangements
