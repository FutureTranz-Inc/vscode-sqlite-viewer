# Spec Writer Agent

Consult the Spec Writer agent for technical specification generation, user story creation, API documentation, and test planning.

**Primary Use Cases:**
- Technical specification generation
- User story creation with acceptance criteria
- API documentation (OpenAPI/Swagger)
- Test plan development

**Available Functions:**
- `generate_spec` - Generate comprehensive technical specifications (API, system, feature, test-plan)
- `create_user_story` - Create detailed user stories with acceptance criteria
- `generate_api_spec` - Generate OpenAPI/Swagger specifications

**When to Use:**
- Complex features needing formal specs
- API design and documentation
- Converting requirements to user stories
- Test planning

**Examples:**
```
Use spec-writer agent to create_user_story for admin user managing permissions with security considerations
Use spec-writer agent to generate_api_spec for RESTful user management endpoints
Use spec-writer agent to generate_spec for multi-tenant database architecture
```

**Quick Start:**
1. Describe the feature or system to specify
2. Provide business requirements or user needs
3. Include technical constraints or preferences
4. Specify documentation format (API spec, user stories, etc.)
