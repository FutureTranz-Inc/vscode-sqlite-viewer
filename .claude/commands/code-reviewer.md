# Code Reviewer Agent

Consult the Code Reviewer agent for comprehensive code quality assessment, performance analysis, and architecture review.

**Primary Use Cases:**
- Code quality assessment
- Security scanning
- Performance analysis
- Architecture review

**Available Functions:**
- `review_code` - Comprehensive code review with quality assessment
- `security_scan` - Security-focused code analysis
- `performance_analysis` - Performance optimization recommendations
- `architecture_review` - Architecture and design pattern analysis

**When to Use:**
- After writing significant code blocks
- Before committing major features
- Performance-critical sections
- Architecture validation

**Examples:**
```
Use code-reviewer agent to review_code for payment processing module with focus on security and performance
Use code-reviewer agent to architecture_review for microservices migration strategy
Use code-reviewer agent to performance_analysis for database query optimization
```

**Quick Start:**
1. Provide the code or module to review
2. Specify focus areas (security, performance, architecture)
3. Include context about the feature/module
4. Mention any specific concerns or requirements
