# Project Manager Agent

Consult the Project Manager agent for sprint planning, task estimation, resource allocation, progress tracking, and risk management.

**Primary Use Cases:**
- Sprint planning and task breakdown
- Time and effort estimation
- Resource allocation
- Progress tracking and reporting
- Risk assessment
- Daily standup facilitation

**Available Functions:**
- `plan_sprint` - Create comprehensive sprint plans with task assignments and estimates
- `estimate_tasks` - Break down features with time/effort estimates
- `assess_risks` - Identify and assess project risks with mitigation strategies
- `track_progress` - Generate progress reports with KPIs and recommendations
- `allocate_resources` - Optimize resource allocation across projects
- `generate_standup` - Generate daily standup agenda and talking points

**When to Use:**
- Starting a new feature or sprint
- Need detailed task estimates
- Managing complex project dependencies
- Tracking multi-task features
- Risk identification

**Examples:**
```
Use project-manager agent to plan_sprint for user authentication feature with team capacity of 40 story points over 2 weeks
Use project-manager agent to estimate_tasks for database migration project with multiple stakeholders
Use project-manager agent to assess_risks for mobile app launch with tight deadline
```

**Quick Start:**
1. Describe the project or sprint scope
2. Provide team capacity and availability
3. Include dependencies and constraints
4. Specify success criteria and deadlines
