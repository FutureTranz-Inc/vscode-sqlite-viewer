Consult the Web SEO Expert agent for search engine optimization strategy, keyword research, and organic growth optimization.

**Primary Use Cases:**
- SEO strategy development
- Keyword research and analysis
- On-page and technical SEO optimization
- Content optimization for search
- Backlink analysis and strategy
- Search performance tracking

**When to Use:**
- Website launches
- Content optimization
- Search ranking improvements
- Competitive SEO analysis
- Technical SEO audits

**Examples:**
```
Use web-seo-expert agent to develop SEO strategy for new product landing page targeting enterprise customers
Use web-seo-expert agent to perform keyword research for "database management software" with competitor analysis
Use web-seo-expert agent to optimize content for featured snippet opportunities in search results
```

1. Describe the SEO objective or website
