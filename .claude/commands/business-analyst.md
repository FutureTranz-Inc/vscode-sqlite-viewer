# Business Analyst Agent

Consult the Business Analyst agent for requirements gathering, process mapping, gap analysis, and stakeholder management.

**Primary Use Cases:**
- Requirements gathering and documentation
- Business process mapping
- Gap analysis (current vs. desired state)
- User story creation from requirements
- Stakeholder analysis and communication planning
- Change impact assessment

**Available Functions:**
- `gather_requirements` - Systematically gather and document business requirements
- `map_process` - Create detailed business process maps and workflows
- `analyze_gaps` - Perform gap analysis between current and desired state
- `create_user_stories` - Convert requirements to user stories with acceptance criteria
- `analyze_stakeholders` - Stakeholder analysis and communication planning
- `assess_change_impact` - Analyze impact of proposed changes on organization

**When to Use:**
- Starting new projects
- Complex business requirements
- Process improvement initiatives
- Stakeholder management
- Organizational change

**Examples:**
```
Use business-analyst agent to gather_requirements for invoice management system with stakeholder input from finance and operations
Use business-analyst agent to map_process for order fulfillment workflow with efficiency optimization
Use business-analyst agent to analyze_gaps between current CRM system and desired sales automation
```

**Quick Start:**
1. Describe the business problem or opportunity
2. Identify key stakeholders and their roles
3. Provide current process documentation
4. Include business objectives and constraints
