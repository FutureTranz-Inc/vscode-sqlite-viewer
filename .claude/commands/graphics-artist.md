# Graphics Artist Agent

Consult the Graphics Artist agent for brand identity design, marketing materials, and visual asset creation.

**Primary Use Cases:**
- Brand identity design
- Marketing materials creation
- Logo design and brand guidelines
- Digital graphics and web assets
- Social media visual content
- Presentation design

**Available Functions:**
- `create_brand_identity` - Design comprehensive brand identity with logo, colors, and typography
- `design_marketing_materials` - Create marketing collateral designs and specifications
- `optimize_visual_design` - Optimize existing designs for better impact
- `create_social_graphics` - Design platform-specific social media graphics
- `develop_brand_guidelines` - Create comprehensive brand style guide

**When to Use:**
- Brand development
- Marketing collateral needs
- Social media content
- Visual identity projects
- Design system creation

**Examples:**
```
Use graphics-artist agent to create_brand_identity for VirtualQ with modern tech personality targeting healthcare industry
Use graphics-artist agent to design_marketing_materials for product brochure with consistent brand application
Use graphics-artist agent to create_social_graphics for LinkedIn post series on AI automation
```

**Quick Start:**
1. Describe the design need or brand project
2. Provide brand context and target audience
3. Include style preferences or existing brand elements
4. Specify usage context (web, print, social)
