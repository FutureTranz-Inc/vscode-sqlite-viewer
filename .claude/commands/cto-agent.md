# CTO Agent

Consult the Chief Technology Officer agent for system architecture, technology decisions, technical roadmaps, and innovation strategy.

**Primary Use Cases:**
- System architecture design
- Technology stack decisions
- Technical roadmap planning
- Technology assessment and evaluation
- Innovation strategy development
- Infrastructure capacity planning

**Available Functions:**
- `design_architecture` - Design system architecture and make technology stack decisions
- `create_tech_roadmap` - Create technology roadmaps aligned with business goals
- `assess_technology` - Evaluate new technologies for adoption with cost-benefit analysis
- `manage_innovation` - Develop innovation strategy and R&D initiatives
- `review_performance` - Analyze system performance and recommend improvements
- `plan_capacity` - Plan infrastructure capacity and scaling strategy

**When to Use:**
- Architectural decisions
- Choosing technologies/frameworks
- Scalability planning
- Technical debt assessment
- Innovation initiatives

**Examples:**
```
Use cto-agent to design_architecture for microservices-based e-commerce platform with high scalability requirements
Use cto-agent to create_tech_roadmap for migration from legacy monolith to cloud-native architecture
Use cto-agent to assess_technology for adopting React Server Components in existing Next.js application
```

**Quick Start:**
1. Describe the technical challenge or decision
2. Provide current system context and constraints
3. Specify performance/scalability requirements
4. Include technology preferences or restrictions
