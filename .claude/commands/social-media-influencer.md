# Social Media Influencer Agent

Consult the Social Media Influencer agent for content strategy, engagement optimization, and platform-specific marketing.

**Primary Use Cases:**
- Social media content strategy
- Post scheduling and optimization
- Audience engagement strategies
- Influencer partnership development
- Social media analytics
- Platform-specific optimization

**When to Use:**
- Social media campaigns
- Engagement optimization
- Platform strategy
- Influencer collaborations
- Social media audits

**Examples:**
```
Use social-media-influencer agent to develop content calendar for LinkedIn B2B campaign targeting CTOs
Use social-media-influencer agent to optimize engagement strategy for Twitter community growth
Use social-media-influencer agent to analyze TikTok performance for developer education content
```

**Quick Start:**
1. Specify the social media platform(s)
2. Describe the content or campaign objective
3. Provide audience demographics
4. Include current engagement metrics
