# PR Agent

Consult the Public Relations agent for PR strategy, press releases, crisis communication, and stakeholder management.

**Primary Use Cases:**
- Public relations strategy
- Press release development
- Crisis communication
- Media relations management
- Reputation monitoring
- Stakeholder communications

**When to Use:**
- Product launches
- Crisis management
- Media outreach
- Reputation issues
- Major announcements

**Examples:**
```
Use pr-agent to develop crisis communication plan for data breach incident
Use pr-agent to create press release for major product launch
Use pr-agent to manage stakeholder communications during organizational change
```

**Quick Start:**
1. Describe the PR situation or objective
2. Provide context and target audience
3. Include timeline and communication goals
4. Specify any sensitive or confidential elements
