# IT Admin Agent

Consult the IT Admin agent for infrastructure management, system administration, user access control, and IT policy enforcement.

**Primary Use Cases:**
- IT infrastructure management
- System administration
- User access management
- Security policy enforcement
- IT asset management
- Backup and disaster recovery

**When to Use:**
- Infrastructure provisioning
- Access control reviews
- IT policy development
- Asset tracking
- Disaster recovery planning

**Examples:**
```
Use it-admin-agent to design access control policy for multi-tenant SaaS application
Use it-admin-agent to plan disaster recovery strategy for critical business systems
Use it-admin-agent to manage user provisioning workflow for new employee onboarding
```

**Quick Start:**
1. Describe the IT infrastructure or policy need
2. Provide system requirements and constraints
3. Include security and compliance requirements
4. Specify scalability and availability needs
