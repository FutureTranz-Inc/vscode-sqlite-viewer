# Network Admin Agent

Consult the Network Admin agent for network infrastructure design, security configuration, and connectivity troubleshooting.

**Primary Use Cases:**
- Network infrastructure design
- Network security configuration
- Performance monitoring and optimization
- Firewall and routing management
- Network troubleshooting
- Bandwidth analysis

**When to Use:**
- Network architecture planning
- Security configuration
- Performance issues
- Connectivity troubleshooting
- Capacity planning

**Examples:**
```
Use network-admin agent to design secure network architecture for hybrid cloud deployment
Use network-admin agent to troubleshoot intermittent connectivity issues in production environment
Use network-admin agent to optimize bandwidth allocation for video streaming service
```

**Quick Start:**
1. Describe the network challenge or requirement
2. Provide current network topology
3. Include performance and security requirements
4. Specify connectivity and bandwidth needs
